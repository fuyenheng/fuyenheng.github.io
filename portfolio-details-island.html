Island for Open World Environment Design

An island I made using Unreal Engine 5. All 3D assets are taken from Quixel Bridge. In this project, I used the landmass and water plugins to create the landscape, river, lakes and ocean. There are river flowing from a hill, and split it to a lake. Another river eventually split into two as well. The details including seashells, ferns, seaweed and some woods. There is no game mechanic in this game. The purpose of this project is to build an island layout map for OpenWorld game.
